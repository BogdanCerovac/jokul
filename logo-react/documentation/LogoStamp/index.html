LogoStamp

React packageStyle package

Se på GitHub

Logoen, logosymbolet og logostemplene våre er merkevareelementer vi kan bruke i Fremtinds løsninger, eller i andre løsninger der Fremtind skal være synlig.

Du kan lese mer om hvordan vi skal bruke logo og logosymbol, under Prinsipper for bruk. Logoen og logosymbolet skal ikke stå synlige sammen. Vi kan animere dem, slik at vi ser at det skifter mellom logo og logosymbol. Se eksempelet.

Vi kan bruke logostemplene der det er behov, for å vise at Fremtind er en del av budskapet. Stemplene skal ikke brukes på samme måte som logoen eller logosymbolet.
